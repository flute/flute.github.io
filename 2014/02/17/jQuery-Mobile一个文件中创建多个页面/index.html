jQuery Mobile一个文件中创建多个页面

在 jQuery Mobile，您可以在单一 HTML 文件中创建多个页面。只需通过唯一的 id 来分隔每张页面，并使用 href 属性来连接彼此即可：

welcome to my blog

Welcome!

This is page one

turn to page two

footer

welcome to my blog

Goodbye!

This is page two

turn to page one

footer
